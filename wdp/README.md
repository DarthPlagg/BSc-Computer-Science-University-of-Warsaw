# 💻 Introduction to Programming (WDP)

My solutions to laboratory assignments and final projects from the "Introduction to Programming" course.

## 📝 Assignment List

1. **Arithmetic Sets (Zbiory arytmetyczne)** – Set-theoretic operations on sets represented as arithmetic progressions. (C)
2. **Three Distinct (Trzy różne)** – An algorithmic problem solved using the "two pointers" (caterpillar) technique. (C)
3. **Origami** – Simulating paper folding within a standard 2D coordinate system. (C)
4. **Sacks (Worki)** – Pointer-based data restructuring and constant-time optimization. (C++)
5. **Strict Intervals (Ścisłe przedziały)** – An algorithmic problem solved using a monotonic queue. (C++)
6. **Pouring Water (Przelewanka)** – An exponential-time solution to an NP-complete problem. (C++)

## 🛠 Compilation & Usage

### For C projects:
```bash
gcc -Wall -Wextra -std=c11 filename.c -o program
```
